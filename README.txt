# Dice of Fate @DreiiCoffee

Works on:

Windows (yes only windows..)

Modern 3D dice UI (Three.js) embedded in a native window.

## Run
 Double-click `run.bat`

First run will install `pywebview` if needed. If the embedded window cannot start, it will open in your default browser. (ITS NOT INSTALLING A VIRUS TO YOUR COMPUTER!!!)


## Customize
- Edit silly fate messages in `web/app.js` under `FATE_MESSAGES`.
- The 666 roll triggers smth
